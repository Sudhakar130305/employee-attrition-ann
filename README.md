# Employee Attrition Prediction using Neural Networks

## Overview

This project predicts employee attrition using an Artificial Neural Network built from scratch with NumPy. The objective is to understand the complete working of neural networks, including forward propagation, backpropagation, gradient descent, and optimization, before using deep learning frameworks like TensorFlow.

---

##  Objectives

- Build an ANN from scratch using NumPy
- Understand every component of a neural network
- Compare the scratch implementation with TensorFlow
- Gain practical deep learning knowledge for AI/ML interviews

---

## 🛠 Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

##  Dataset

IBM HR Analytics Employee Attrition & Performance Dataset (Kaggle)
